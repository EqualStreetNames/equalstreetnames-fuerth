# EqualStreetNames Fürth

This repository contains the [EqualStreetNames](https://equalstreetnames.eu/) submodule für Fürth in Bavaria, Germany.

The map at [fuerth.equalstreetnames.eu](https://fuerth.equalstreetnames.eu) shows the streets and places in Fürth in Bavaria, Germany, tagged with `name:etymology:wikidata`
in [OpenStreetMap](https://www.openstreetmap.org) resp. in the file data.csv, and visualizes the **gender-distribution** of their names.
